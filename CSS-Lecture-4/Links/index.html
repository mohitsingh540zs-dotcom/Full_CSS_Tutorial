Links

Links are one of the most important elements of any website. By default, browsers give them a standard blue color and an underline.

Different Link States

  • a:link

    Styles a normal, unvisited link

  • a:visited

    Styles a link that the user has already clicked.

  • a:hover

    Styles a link when the mouse pointer is placed over it.

  • a:active

    Styles a link when it is being clicked.

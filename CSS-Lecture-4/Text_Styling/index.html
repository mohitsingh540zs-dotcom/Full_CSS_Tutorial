Text Styling

Text styling in CSS allows you to control how text appears on a web pages.With a few properties, we can decorate text, control its alignment, transform its case, adjust spacing, and even add effects like shadows.

Text Decoration

The text-decoration property adds various types of text decorations.

Syntax:- 
        selector:{

            text-decoration:value;

        }
Example:- Mohit Google

Text Alignment

The text-align property is used to align the text within the container. The container can be a div, section, etc..

Syntax:- 
        selector:{

            text-align:value;
            
        }
Example:-
Hello World

Text Transformation

The text-transform property changes the case of the text.

Syntax:- 
        selector:{

            text-transform:value;
            
        }
Example:-

Mai hu Bummble Bee

Letter Spacing

The letter-spacing property allows us to specify the spacing between each character in the text.

Syntax:- 
        selector:{

            letter-spacing:value;
            
        }
Example:-

HelloWorld

Line Height

The line-height property defines the vertical spacing between line of text.

Syntax:- 
        selector:{

            line-height:value;
            
        }
Example:-
Text-1
Text-2

Text Shadows

The text-shadow property adds a shadow effect to text.

Syntax:- 
        selector:{

            text-shadow:value;
            
        }
Example:-

Hello I'm Mohit

Text Overflow

text-overflow property controls how overflowing text is handled inside a container.

Syntax:- 
        selector:{

            text-overflow:value;
            
        }

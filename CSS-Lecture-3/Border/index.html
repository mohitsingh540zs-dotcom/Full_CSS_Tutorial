Border

Borders in CSS define the visual boundaries of HTML elements such as div, p, section, article, etc..

Border Styles

Border style defines the style of border/boundary of html elements.

Syntax:- 
        selector{
            border: size style color;
        }

Types of border styles

  1. none
  2. hidden
  3. dotted
  4. dashed
  5. double
  6. solid
  7. groove
  8. ridge
  9. inset
  10. outset

Border Radius

Border radius helps to create rounded borders for elements like button and image.

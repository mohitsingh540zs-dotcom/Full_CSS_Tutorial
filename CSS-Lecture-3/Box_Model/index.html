Box-Model

The CSS Box model defines how elements are rendered and how their dimensions are calculated. It describes the stucture of an element as a rectangular box that has content, padding, border, and margin.

Components of Box model

  1. Content
  2. Padding
  3. Border
  4. Margin

Total Width = Width + Left Padding + Right Pading + Left Border + Right Border + Left Margin + Right Margin

Total Height = Height + Top Padding + Bottom Pading + Top Border + Bottom Border + Top Margin + Bottom Margin

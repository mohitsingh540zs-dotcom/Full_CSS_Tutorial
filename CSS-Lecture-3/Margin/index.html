Margin

The margin is the space outside an HTML element. It provides external spacing and helps to separate to or more elements from each other.

Ways of applying margins

  1. Individual Margin

    Syntax:-

                    selector{
                margin: Top Right Bottom left;
            }
  2. Vertical & Horizontal Margin

    Syntax:-

                    selector{
                    margin: vertical horizontal;
                }
  3. All sides Margin

    Syntax:-

                    selector{
                    margin: value;
                }

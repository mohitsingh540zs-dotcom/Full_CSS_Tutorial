Ways of appling padding

  1. Individual padding

    Syntax:-

    selector{
                padding: Top Right Bottom left;
            }
  2. Vertical & Horizontal padding

    Syntax:-

                    selector{
                    padding: vertical horizontal;
                }
  3. All sides padding

    Syntax:-

                    selector{
                    padding: value;
                }

Position

The CSS Position property allows us to control how and where elements are placed on a web page. it defines whether the element should follow the natural flow of document or be positioned at a specidic spot using coordinates like top, bottom, left and right.

Possible values of postion

  • static: Default positioning (noraml document flow)
  • relative: Positioned relative to its normal postion.
  • absolute: Position relative to nearest positioned ancestor.
  • fixed: Positioned relative to the viewport (stays fixed while scrolling too).
  • sticky: Acts like relative until a scroll threshold is reached, then sticks like fiexed.

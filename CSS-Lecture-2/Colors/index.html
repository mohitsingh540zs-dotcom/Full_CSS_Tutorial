Hexadecimal notation

The hex code consits of a # hash symbol followed by 6 characters.

These 6 characters are arranged into set of three pairs (RR, GG, and BB)

R -> Red, G -> Green, B -> Blue

The intensity lies between 00 and ff

  • RR

    RR defines the intensity of color red, ranging from 00 to FF

  • GG

    GG defines the intensity of color green, ranging from 00 to FF

  • BB

    BB defines the intensity of color blue, ranging from 00 to FF

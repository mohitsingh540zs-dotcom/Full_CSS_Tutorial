Selectors

CSS selectors allow us to choose specific elements and apply styles to them.

There are different types of css selectors

  1. Universal Selector
  2. Element Selector
  3. Id Selector
  4. Class Selector
  5. Group Selector

#1 Universal Selector

Universal selector is represented by * targets all the HTML elements on the page.

Syntax:- 
        * {

            property: value;

        }

#2 Element Selector

The element selector selects the target element based on specific type.

Syntax:- 
        element {

            property: value;

        }

#3 ID Selector

The ID selector targets the elements based on the specific id.

Syntax:- 
        #ID {

            property: value;

        }
Important

#4 Class Selectors

The class selector does the same job as id selector, a class selector helps group various types of elements.

Syntax:- 
        .className{

            property: value;

        }

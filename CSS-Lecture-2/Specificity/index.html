# Specificity

When you style a webpage, sometimes multiple CSS rules target the same element. So, which one wins? That's where the CSS specificity comes in.

it's basically a weight system that determines which rule has more power.

Mohit

The Cascade Algorithm

The cascade algorithm is used for solving conflicts where multiple css rules apply to an HTML element

  1. Postion and order of appearance:-

    The order in which your css rules appear.

  2. Specificity

    an algorithm that determines which css selector has the strongest match (Inline > Id > Class > Element > Universal).

  3. Origin:-

    the order in which css apears and where it comes from, whether that is a browser style, css from a browser extension, or your authorized css.

  4. Importance:-

    some css rules are weighted more heavily than others, especially with the !important rule type.
